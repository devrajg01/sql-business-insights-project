# SQL Business Insights Project

## Objective
To analyze sales, customer, and product data using SQL and generate actionable business insights.

## Database Structure
- customers
- products
- orders

## Analysis Performed
- Product-wise sales performance
- Region-wise sales contribution
- Top customers by revenue
- Monthly sales trend analysis
- Category-wise profitability analysis
- Profit classification using CASE statements

## SQL Concepts Used
- JOIN
- GROUP BY
- HAVING
- ORDER BY
- CASE
- Aggregate functions

## Tools
- MySQL

## Key Insights
- Technology category generates the highest revenue and profit
- West region contributes the maximum sales
- Repeat customers drive significant revenue
